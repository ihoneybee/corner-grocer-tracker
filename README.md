# cpp-project-3-vector-sort
This project is a C++ item-tracking program. The program reads a text file of purchased items and analyzes how frequently each item appears.
I did well organizing the program using a class, separating functionality into clear methods, and correctly using a map to track item frequencies.
I could improve the program by adding input validation, handling file errors better, and making the menu more user-friendly.
The most challenging part was working with file input/output and correctly counting item frequencies, which I overcame by testing each function and using examples from class.
This project strengthened my skills in file handling, using maps, writing classes, and building programs with menus.
I made the program maintainable and readable by using a class structure, using clear function names, and separating input, processing, and output into different parts of the code.
